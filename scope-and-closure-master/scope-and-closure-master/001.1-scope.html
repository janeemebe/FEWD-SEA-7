Function Scope

Let's test out function scope! Complete steps 1 and 2, and we'll walk through the rest together.

Every variable defined in a function, is available for the entirety of that function.

  1. Open js/001.1-scope.js in Sublime Text.
  2. Add a variable inside the function testFunctionScope on line 5. Name it whatever you like, and set it to one of the available data types.
  3. Uncomment the debugger statements on line 6.
  4. Make sure your developer tools are open in Chrome, and refresh the page.
  5. The page should stop executing JavaScript when it reaches that dubugger statement. You can open the console by clicking the drawer icon:
    • (blue icon on the left)
  6. Your console is now inside the the testFunctionScope function - frozen in time. Try accessing the variables inside that function.
  7. Uncomment the debugger statement on line 10. Try accessing the variables before the function call, then continue to the debugger statement inside the function.
  8. Uncomment the debugger statement on line 12. Try accessing the variables after the function call.
