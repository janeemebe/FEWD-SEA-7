  1. Open project/js/instasearch.view.js in Sublime Text.
  2. Delete the sampleImageObject from this file.
  3. Create a new variable named sampleArray and assign it the value above (no quotations needed).
    • The text block above is a minified version of a JavaScript Array. Don't worry if the formatting is hard to understand.
  4. Add a function named formatResults to project/js/instasearch.view.js.
    • Don't worry about Closure at this point - put it at the same level/scope as imageTemplate.
    • formatResults function will accept one parameter (the Array of objects - sampleArray).
    • Don't store your function as a variable - just declare it directly: 
      function formatResults(dataArray) {}
  5. Inside the Scope of formatResults, create a variable named resultsHTML and assign it the value of an empty string ('').
  6. Loop through the Array stored under your parameter name. For each iteration of your loop, append more results to the resultsHTML variable by calling the imageTemplate function with each Array value: 
    resultsHTML += imageTemplate(arrayName[i]);
  7. Return the resultsHMTL variable at the end of your formatResults function.
  8. At the bottom of the instasearch.view function, store the result of the formatResults in a variable: 
    var results = formatResults(sampleArray);
  9. Find the container for the results in project/instasearch.html. Select it with jQuery, and then insert the results using the jQuery function html().

Hints:

  • You can select items with jQuery using the $ symbol and then the CSS selector. For example, to select a DIV with the ID myID, you'd write: 
    $("#myID")
  • You can chain jQuery functions after selecting an item. For example: 
    $("#myID").hmtl(results);
  • Add a debugger statement or breakpoint if you are having issues.
  • Check the console for errors if you don't see any results.
