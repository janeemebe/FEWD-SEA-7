Global Scope

Let's play with Global Scope one more time.

Fool me once, shame on you. Fool me twice, shame on me.

  1. Open js/001.3-scope.js in Sublime Text.
  2. On line 7, assign the the global variable name to the parameter workshopName.
  3. On line 17, assign the the global variable name to the parameter studentName.
  4. In the console, set the student name using the setStudentName function of the student object: 
    student.setStudentName(<your name>);
  5. In the console, log the student name using the logStudentName function of the student object: 
    student.logStudentName();
  6. In the console, set the workshop name using the setWorkshopName function of the workshop object: 
    workshop.setWorkshopName("Intermediate JavaScript & jQuery");
  7. In the console, log the workshop name using the logWorkshopName function of the workshop object: 
    workshop.logWorkshopName();
  8. In the console, try to log the student name again by repeating step 5.
