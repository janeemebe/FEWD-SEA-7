Closure

Let's shadow and reassign some variables in scope.

  1. Open js/003.2-closure.js in Sublime Text.
  2. Review the functions, the log statements, and what is being logged in the console.
  3. Shadow the variable a inside the bar function.
    • Hint: Create a local variable named a on line 10, and set it to a different value than 2.
  4. Reassign the variable b inside the bar function.
    • Hint: Assign b on line 13 to a different value than 4.
  5. Check whether or not the log statements are different in the console.
  6. Check if assigning b a value without the var keyword created a global variable by typing the variable b in the console.
