Lab: InstaSearch

Phase 3: Let's stop using mock data, and fetch some real data via AJAX!

InstaSearchStarter Code

AJAX Get Syntax:

$.get(url, function(jsonData) {
  // do something with our
  // our jsonData here
}, "jsonp");


  1. Open project/js/instasearch.model.js in Sublime Text.
  2. Create a local variable names url and assign it to the result of the instagramEndpoint function on line 11.
  3. Add your AJAX function, using the $.get() method on line 13. Inside the success callback, call handleResults with your data as it's parameter.
    • Make sure you are passing just the array of images, and not the whole object: 
      handleResults(jsonData.data);
  4. Inside the handleResults function, let's invoke our view functions by adding the line: 
    instasearch.view(dataArray);
  5. Let's stop calling the view functions manually:
    1. Open project/js/instasearch.view.js in Sublime Text.
    2. Delete line 34 which invokes the view function: 
      instasearch.view();
    3. Delete the sampleArray declaration.
    4. Replace any references to sampleArray with resultsArray - the parameter passed into the instasearch.view function.
  6. Let's test our model and view functions! Below the comment // add your test search here, call the fetchData function with a tag you want to search for: 
    fetchData('puppy');
