Global Scope

Let's mess with the Global Scope!

Everything not defined inside a scope (i.e. function scope) will become a global variable.

  1. Open js/001.2-scope.js in Sublime Text.
  2. On line 2, define a global variable named location and set it to null:
    location = null;
    Note: After making the change and refreshing the page, uncomment line 2 and hit back in your browser.
  3. Repeat step 2, but inside the myFunction on line 7.
  4. On line 5, declare a local variable named location, but don't assing it a value: 
    var location;
  5. Uncomment line 7 which assigns the local variable location to null.
