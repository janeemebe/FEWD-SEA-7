Lab: InstaSearch

Phase 1: Using our knowledge of scope, let's write a templating function with the sample object provided.

InstaSearch

HTML Tempalte:

<div class="col-sm-6 col-md-4">
    <div class="thumbnail">
        <img src="<IMAGE URL>">
    </div>
</div>


  1. Open project/instasearch.html and project/js/instasearch.view.js in Sublime Text.
  2. Add a function named imageTemplate on line 28 of project/js/instasearch.view.js.
    • The function should accept one parameter - an object that contains the image information.
    • There is an example image object that we will pass to our imageTemplate function in the JavaScript file.
    • Don't store your function as a variable - just declare it directly: 
      function imageTemplate(imagesObject) {}
  3. Create a variable and assign it the url of the standard_resolution version of our image.
  4. Contstruct another variable that will hold all the HTML that imageTemplate will return. See the example HTML Template above.
  5. Make sure that you replace <IMAGE URL> with the url variable we created.
  6. At the end of our imageTemplate function, return that variable that holds our final HTML: 
    return variableName;
  7. Test our function by calling it at the bottom of our JavaScript file. Pass it our sample image object, and log the result: 
    console.log(imageTemplate(sampleImage));

Hints:

  • You can combine two strings with the concatenation operator (+): 
    var result = "string 1" + variableString;
  • You can navigate through an object using the dot syntax: 
    grandfatherObject.parentObject.childObject.variable
  • Add a debugger statement or breakpoint in console inside the instasearch.view function, and test navigating the sampleImage object: 
    sampleImage.thumbnail.height
