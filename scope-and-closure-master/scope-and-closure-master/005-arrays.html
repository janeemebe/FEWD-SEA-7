Arrays

Create an array and loop through it.

Example Array:

var cars = ["Saab", "Volvo", "BMW"];

Example for statement:

for (var i = 0; i < 9; i++) {
   console.log(i);
   // more statements
}
  1. Open js/005-arrays.js in Sublime Text.
  2. Create an variable named favoriteMovies and assign it the value of an Array of your favorite movies.
  3. Write a for statement that loops through your array.
    • Hint: In the example above, replace i < 9 with favoriteMovies.length.
  4. For the statement of the for loop, log the value of the Array at position i to the console: 
    console.log(favoriteMovies[i]);
